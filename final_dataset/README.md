# Final Dataset

This directory contains manually reviewed, accepted samples intended for final
research delivery.

Current accepted sources:

- `apache_commons_lang/`: 37 high-quality Javadoc evolution samples.

`combined_samples.json` aggregates all accepted samples across sources. New
mining runs should remain in source-labelled temporary directories such as
`dataset_<source>_<count>` until they have been reviewed and accepted.
